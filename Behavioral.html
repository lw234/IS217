Behavioral Pattern:are design patterns that identify common communication patterns between objects and realize these patterns. By doing so, these patterns increase flexibility in carrying out this communication. The Interpreter pattern defines a grammatical representation for a language and an interpreter to interpret the grammar. Musicians are example of Interpreters. The pitch of a sound and its duration can be represented in musical notation on a staff. This notation provides the language of music.

Implement Prototype and Strategy Patterns

The second example the pattern that I selected to implement is prototype and strategy pattern. Prototype is a creational design pattern and strategy is a behavioral design pattern. In this example, I used prototype pattern to create Car function. The prototype pattern help improve on the method creation by appending them all at once to the prototype within an object literal. For this example the strategy pattern is use to help Car function to greet different car owner differently. In order to do that, I created different Strategies to say hello. The greeting function provides a great way to encapsulate an algorithm and can be used as a strategy.
