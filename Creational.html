Creational Pattern: are design patterns that deal with object creation mechanism, trying to create objects in a manner suitable to the situation. Creational patterns focus on ways to create objects or classes. The basic form of object creation could result in design problems or added complexity to the design. Creational design patterns solve this problem by somehow controlling this object creation. Implement Factory and Decorator Pattern

The first example the pattern that I selected to implement is factory and decorator pattern. Factory is a creational design pattern and decorator is a structural design pattern. In this example, I used factory pattern to create Frozen Yogurt object because by using factory pattern you can create objects without the need to repeat yourself. For decorator pattern, it is used for adding different kinds of topping to the object of Frozen Yogurt and the cost of each topping to the starting cost of the Frozen Yogurt. Basically, a decorator allows customers to add new topping to Frozen Yogurt object without the need to modify the entire code structure.
